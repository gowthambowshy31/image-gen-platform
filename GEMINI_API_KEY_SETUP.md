# How to Get Your Gemini API Key

Follow these simple steps to obtain your Google Gemini API key:

## Step 1: Visit Google AI Studio
Go to [Google AI Studio](https://aistudio.google.com/)

## Step 2: Sign In
- Sign in with your Google account
- If you don't have a Google account, you'll need to create one first

## Step 3: Access API Keys
- Once logged in, look for the **"Get API Key"** button in the top navigation or sidebar
- Click on it to access the API keys section

## Step 4: Create a New API Key
1. Click on **"Create API Key"** button
2. You may be prompted to:
   - Select or create a Google Cloud project (you can use the default project)
   - Accept terms of service if this is your first time
3. Your API key will be generated instantly

## Step 5: Copy Your API Key
- Copy the API key that appears (it will look like: `AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXX`)
- **Important**: Store this key securely - treat it like a password

## Step 6: Share the API Key
Send us your API key through a secure channel

## Important Notes

- **Free Tier**: Gemini API offers a free tier with generous usage limits
- **Security**: Never share your API key publicly or commit it to version control
- **Billing**: Monitor your usage at [Google Cloud Console](https://console.cloud.google.com/)
- **Rate Limits**: Free tier has rate limits - check [current pricing](https://ai.google.dev/pricing) for details

## Troubleshooting

**Can't see the "Get API Key" button?**
- Make sure you're logged into your Google account
- Try accessing directly: https://aistudio.google.com/app/apikey

**API key not working?**
- Verify you copied the entire key
- Check if API key restrictions are enabled (not recommended for initial testing)
- Ensure the Gemini API is enabled for your project
